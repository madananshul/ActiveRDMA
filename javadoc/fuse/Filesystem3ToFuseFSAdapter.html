  • java.lang.Object

public class Filesystem3ToFuseFSAdapter
extends java.lang.Object
implements FuseFS
This is an adapter that implements fuse.FuseFS byte level API and delegates to the fuse.Filesystem3 String level API. You specify the encoding to be used for file names and paths.
