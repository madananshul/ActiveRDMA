public class Filesystem2ToFilesystem3Adapter
extends java.lang.Object
implements Filesystem3
This is an adapter that adapts the semi-old filehandle enabled API (fuse.compat.Filesystem2) to the new errno-as-return-value-or-exception API (fuse.Filesystem3)
